# Md-Aulad-Hossain-Sikder-
৩০ রোযা উপলক্ষে ৩০ টাকা মোবাইল রিচার্জ 🎉 আমার ফেসবুক প্রোফাইল ফলো ও শেয়ার করলেই পাচ্ছেন। 🎁
নিয়ম হলো ✅ আপনার নাম এবং ✅ ফোন নাম্বার দিবেন। এরপর ✅ আমার ফেসবুক প্রোফাইল টিকে ফলো করবেন এরপর ✅ আপনার ৪ ফ্রেন্ডের মাঝে এটা শেয়ার করবেন। 
কোন ভয় নেই আমার সাথে আপনার 📲 ভিডিও কলে কথা হবে।
